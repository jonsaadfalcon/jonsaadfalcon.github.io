# Website Inspirations

Sites that informed the design of this personal website. The combined goal: a polished, content-first academic site with a Georgia + Forest Mint visual identity, suitable for the dual audience of faculty hiring committees and industry/investors.

## Primary inspirations (used heavily)

- **Shreya Shankar** — [sh-reya.com](https://www.sh-reya.com/) — Section structure (representative papers separate from the full list, mentee display, job-market materials prominent), clean hierarchy. The mentee section format ("Advisor / Current Mentees / Past Mentees" with parenthetical destinations) is taken from her site.
- **Nicholas Carlini** — [nicholas.carlini.com](https://nicholas.carlini.com/) — Typographic identity: serif font (Lora) plus a single accent color (burgundy). We adapted this with Georgia + Forest Mint. Animated link underlines and personal voice throughout.
- **Audrey Cheng** — [audreyccheng.com](https://audreyccheng.com/) — Fluid `clamp()` typography, two-column hero with portrait, news timeline with fixed date column, talks formatted as "Title" + nested venue+date list, awards section with year right-aligned.

## Secondary inspirations

- **Niklas Muennighoff** — [muennighoff.com](https://muennighoff.com/) — Horizontal scrollable research cards, semantic color-coded category tags. Considered but not used; would require more visual assets per project than we currently have.
- **Sewon Min** — [sewonmin.com](https://www.sewonmin.com/) — Clean academic structure, collapsible info boxes for prospective students. Considered for future expansion.
- **Omar Khattab** — [omarkhattab.com](https://omarkhattab.com/) — Inline project logo treatment (DSPy, ColBERT). Idea reserved for future use if/when standalone projects warrant.
